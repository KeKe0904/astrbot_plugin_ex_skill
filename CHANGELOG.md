# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2026-04-06

### Added
- 初始版本发布
- 实现创建前任 Skill 功能
- 实现列出前任 Skill 功能
- 实现删除前任 Skill 功能
- 实现放下前任功能（删除的温柔别名）
- 支持多数据源：微信聊天记录、QQ消息、照片等
- 支持智能性格分析：基于 MBTI、星座、依恋类型等
- 支持关系记忆提取：自动分析共同经历、约会地点等
- 支持进化机制：追加记忆、对话纠正、版本管理
- 跨平台支持：支持多种消息平台

### Changed
- 插件名称改为 astrbot_plugin_ex_skill
- 作者信息更新为落梦陳

### Fixed
- 修复了插件初始化时的目录创建问题
- 优化了命令处理逻辑
