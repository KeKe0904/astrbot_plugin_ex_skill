# Changelog

All notable changes to this project will be documented in this file.

## [1.0.6] - 2026-04-06

### Added
- 实现完整的前任 Skill 调用功能
- 添加三种调用模式：完整模式、记忆模式、性格模式
- 支持用户与前任 Skill 对话
- 添加回复风格模拟，使回复更符合前任的性格特点
- 优化 slug 处理，确保生成有效的目录名

### Fixed
- 优化消息处理逻辑，确保对话流程顺畅
- 增强错误处理，提高插件稳定性

## [1.0.5] - 2026-04-06

### Added
- 实现完整的对话交互功能
- 添加消息事件处理，支持分步骤收集用户输入
- 添加前任 Skill 生成功能，自动创建 memory.md 和 persona.md 文件
- 添加版本管理，自动备份初始版本
- 支持用户跳过步骤

### Fixed
- 修复了消息编号显示问题，使用中文数字格式
- 修复了标点符号问题，移除多余的逗号

## [1.0.4] - 2026-04-06

### Fixed
- 修复了命令处理函数的参数定义问题
- 确保所有指令使用正确的参数格式

## [1.0.3] - 2026-04-06

### Changed
- 取消支持 weixin_official_accounts 和 atori 平台
- 添加支持 satori 平台

## [1.0.2] - 2026-04-06

### Added
- 添加启动自检功能，确保插件启动时正常运行
- 检查运行平台、插件配置、目录结构、前任 Skill 列表
- 检查文件权限、依赖项和平台支持
- 添加详细的启动日志，便于排查问题

## [1.0.1] - 2026-04-06

### Added
- 添加平台兼容性支持，确保在所有平台上正常工作
- 添加平台检测和平台特定提示信息
- 完善跨平台文件路径处理

### Changed
- 优化日志记录，添加平台信息
- 改进用户提示，根据不同平台提供不同的建议

## [1.0.0] - 2026-04-06

### Added
- 初始版本发布
- 实现创建前任 Skill 功能
- 实现列出前任 Skill 功能
- 实现删除前任 Skill 功能
- 实现放下前任功能（删除的温柔别名）
- 支持多数据源：微信聊天记录、QQ消息、照片等
- 支持智能性格分析：基于 MBTI、星座、依恋类型等
- 支持关系记忆提取：自动分析共同经历、约会地点等
- 支持进化机制：追加记忆、对话纠正、版本管理
- 跨平台支持：支持多种消息平台

### Changed
- 插件名称改为 astrbot_plugin_ex_skill
- 作者信息更新为落梦陳

### Fixed
- 修复了插件初始化时的目录创建问题
- 优化了命令处理逻辑
